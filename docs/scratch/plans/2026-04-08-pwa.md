> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the app installable on an iPhone home screen so it opens full-screen without Safari browser chrome.

**Architecture:** Three changes only — generate PNG icons, add `manifest.json` to `client/public/`, and add Apple PWA meta tags to `client/index.html`. Vite serves `public/` as static assets automatically, so no build config changes needed.

**Tech Stack:** Node.js `canvas` package for icon generation, standard Web App Manifest JSON, HTML meta tags.

- [ ] **Step 2: Create the icon generation script**

Create `/Users/waled/Desktop/mudhiyan-workshop/generate-icons.js`:

```js
const { createCanvas } = require("canvas");
const fs = require("fs");
const path = require("path");

const outDir = path.join(__dirname, "client/public/icons");
fs.mkdirSync(outDir, { recursive: true });

function drawIcon(size) {
  const canvas = createCanvas(size, size);
  const ctx = canvas.getContext("2d");

  // Background
  ctx.fillStyle = "#16120D";
  ctx.fillRect(0, 0, size, size);

  // Gold circle border
  const cx = size / 2;
  const cy = size / 2;
  const r = size * 0.42;
  ctx.beginPath();
  ctx.arc(cx, cy, r, 0, Math.PI * 2);
  ctx.strokeStyle = "#C9A84C";
  ctx.lineWidth = size * 0.04;
  ctx.stroke();

  // Gold ◈ symbol
  ctx.fillStyle = "#C9A84C";
  ctx.font = `bold ${size * 0.42}px serif`;
  ctx.textAlign = "center";
  ctx.textBaseline = "middle";
  ctx.fillText("◈", cx, cy);

  return canvas.toBuffer("image/png");
}

fs.writeFileSync(path.join(outDir, "icon-180.png"), drawIcon(180));
fs.writeFileSync(path.join(outDir, "icon-512.png"), drawIcon(512));
console.log("Icons generated: icon-180.png, icon-512.png");
```

## Task 3: Add Apple PWA meta tags to index.html

**Files:**

- Modify: `client/index.html`

- [ ] **Step 1: Replace the entire `<head>` section**

Replace the current `<head>` block in `client/index.html` with:

```html
<!doctype html>
<html lang="ar" dir="rtl">
  <head>
    <meta charset="UTF-8" />
    <meta
      name="viewport"
      content="width=device-width, initial-scale=1.0, viewport-fit=cover"
    />
    <title>مجوهرات سليمان المضيان</title>

    <!-- PWA Manifest -->
    <link rel="manifest" href="/manifest.json" />

    <!-- iOS PWA -->
    <meta name="apple-mobile-web-app-capable" content="yes" />
    <meta
      name="apple-mobile-web-app-status-bar-style"
      content="black-translucent"
    />
    <meta name="apple-mobile-web-app-title" content="المضيان" />
    <link rel="apple-touch-icon" href="/icons/icon-180.png" />

    <!-- Theme color (Android + browser tab) -->
    <meta name="theme-color" content="#C9A84C" />

    <!-- Fonts -->
    <link rel="preconnect" href="https://fonts.googleapis.com" />
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
    <link
      href="https://fonts.googleapis.com/css2?family=Almarai:wght@300;400;700;800&family=JetBrains+Mono:wght@400;600&display=swap"
      rel="stylesheet"
    />
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/src/main.jsx"></script>
  </body>
</html>
```
